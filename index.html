Tailwind CSS is a utility-first CSS framework that gives you low-level classes to build completely custom designs — without ever leaving your HTML.

Get Started See an Example

What is Tailwind CSS?

Unlike traditional CSS frameworks like Bootstrap, Tailwind doesn't provide pre-built components. Instead, it gives you a huge set of small, single-purpose utility classes that you compose to build your own designs.

You style elements directly in HTML using classes like flex, pt-4, text-center, and rotate-90 — no context switching to a CSS file.

At build time, Tailwind scans your HTML and generates only the CSS you actually use — resulting in tiny, production-ready stylesheets.

❌ Traditional CSS

/* styles.css */
.card {
  padding: 1.5rem;
  border-radius: 1rem;
  background: #1e293b;
  box-shadow: 0 4px 6px rgba(0,0,0,.1);
}

✅ Tailwind CSS

<!-- HTML -->
<div class="p-6 rounded-2xl
       bg-slate-800 shadow-md">
  ...
</div>

Why Tailwind?

Everything you need to build modern UIs — fast.

⚡

No CSS to Write

Style everything directly in your HTML. No back-and-forth between files. Your markup and style live together.

📦

Tiny Bundle Size

Tailwind scans your files and strips unused styles at build time. Most projects ship less than 10 kB of CSS.

🎨

Fully Customizable

Extend the design system with your own colors, fonts, spacing, and more via a simple config file.

📱

Responsive by Default

Every utility can be prefixed with a breakpoint: sm:, md:, lg:. Mobile-first, always.

🌙

Dark Mode Ready

Use the dark: prefix to style elements differently in dark mode. Works with class or media query strategies.

🔌

Plugin Ecosystem

Official plugins for typography, forms, and aspect-ratio. Plus a rich community of third-party plugins.
